Input for creating a post

```graphql
type CreatePostInput {
  attachmentIds: [String!]
  mappingFields: [PostMappingFieldInput!]
  postTypeId: String!
  publish: Boolean
  tagNames: [String!]
}
```

### Fields

#### `attachmentIds` ([`[String!]`](/graphql/scalars/string))

The id of all the attachments to this post.

#### `mappingFields` ([`[PostMappingFieldInput!]`](/graphql/inputs/post-mapping-field-input))

The fields of the post. Depending of the post type it may include title, text, image, etc.

#### `postTypeId` ([`String!`](/graphql/scalars/string))

The id of this post's type

#### `publish` ([`Boolean`](/graphql/scalars/boolean))

Whether the post is published

#### `tagNames` ([`[String!]`](/graphql/scalars/string))

The tags of this post.
