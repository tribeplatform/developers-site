Input for creating a tag.

```graphql
type CreateTagInput {
  description: String
  slug: String!
  title: String!
}
```

### Fields

#### `description` ([`String`](/docs/graphql/scalars/string))

The description of the tag.

#### `slug` ([`String!`](/docs/graphql/scalars/string))

The slug of the tag.

#### `title` ([`String!`](/docs/graphql/scalars/string))

The name of the tag.
