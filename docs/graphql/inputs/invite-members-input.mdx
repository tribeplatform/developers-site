Input for inviting members.

```graphql
type InviteMembersInput {
  defaultSpacesIds: [ID!]
  expiresAt: DateTime
  invitationMessage: String
  invitees: [InviteeInput!]!
  roleId: ID
}
```

### Fields

#### `defaultSpacesIds` ([`[ID!]`](/graphql/scalars/id))

The ids of the default spaces the invitees have.

#### `expiresAt` ([`DateTime`](/graphql/scalars/date-time))

When does the invitations expire? Empty for no expiration.

#### `invitationMessage` ([`String`](/graphql/scalars/string))

Custom message for invitation.

#### `invitees` ([`[InviteeInput!]!`](/graphql/inputs/invitee-input))

The details of the invitees

#### `roleId` ([`ID`](/graphql/scalars/id))

The id of the role the invitees have.
